## 🚀 La Mission
**E-commerce RDC Pro** n'est pas qu'une simple application ; c'est un écosystème conçu pour propulser les propriétaires de boutiques en République Démocratique du Congo dans l'ère numérique. Notre plateforme automatise la création de boutiques en ligne, permettant aux commerçants locaux de toucher une clientèle nationale tout en offrant une opportunité génératrice de revenus pour la jeunesse congolaise via notre programme de parrainage.

---

## ✨ Points Forts du Système

### 👔 Dashboard Administrateur (Executive Control)
- **Analytics en temps réel**: Surveillance des pistes (prospects), conversions et revenus globaux.
- **Gestion des Partenaires**: Activation manuelle des affiliés et génération de **Codes d'Accès Uniques**.
- **Support Chat Centralisé**: Interface de messagerie directe pour assister les partenaires en temps réel.
- **Validation des Paiements**: Gestion simplifiée des demandes de retrait de commissions.

### 🤝 Portail Partenaire (Affiliate Ecosystem)
- **Onboarding Interactif**: Un parcours visuel expliquant l'impact social et financier de rejoindre le programme.
- **Tracking Précis**: Suivi transparent des prospects générés et des ventes confirmées.
- **Génération de Liens de Paiement**: Outil pour créer des liens de vente personnalisés pour les clients.
- **Profil Personnalisé**: Téléchargement d'avatar et gestion du nom d'affichage.

### 🔐 Sécurité & Intégrité
- **Accès par Code Unique**: Authentification sécurisée via Téléphone + Code fourni par l'admin (pas de mots de passe oubliés).
- **Confirmation Double**: Validation des emails lors de l'inscription pour réduire les erreurs.

## 🛠 Stack Technique
- **Frontend**: [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/) + [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Backend**: [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/)
- **Base de Données**: [SQLite](https://www.sqlite.org/) via [better-sqlite3](https://github.com/WiseLibs/better-sqlite3)
- **Icônes & UI**: Google Material Symbols + Phosphor Icons

## 🏁 Démarrage Rapide

### Prérequis
- [Node.js](https://nodejs.org/) (v16+)
- npm ou yarn

### Installation
1.  **Cloner le projet**
    ```bash
    git clone [repository-url]
    cd E-commerce-RDC-Pro--main
    ```

2.  **Installer les dépendances**
    ```bash
    npm install
    ```

3.  **Lancer l'application**
    Ouvrez deux terminaux :
    - **Terminal 1 (Backend)**: `npm run server`
    - **Terminal 2 (Frontend)**: `npm run dev`
